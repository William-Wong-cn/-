# -基于多模型深度融合的股票价格预测与交易策略回测系统
本项目是一个高性能的时序分析解决方案，旨在通过深度学习和经典机器学习的融合，实现股票价格预测和交易信号生成。项目严格遵循企业级模块化标准，全面展示了从数据清洗、特征工程、双模型训练到量化策略回测的全流程实践能力。
This project is a high-performance time series analysis solution, aiming to achieve stock price prediction and trading signal generation through the integration of deep learning and classical machine learning. The project strictly adheres to enterprise-level modular standards and comprehensively demonstrates the full-process practical capabilities from data cleaning, feature engineering, dual-model training to quantitative strategy backtesting.
